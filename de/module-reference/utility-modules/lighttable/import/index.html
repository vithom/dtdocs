    Import

    Importiere Bilder in eine oder mehr als eine Filmrolle.

    Bilder können ab einem lokalen Verzeichnis oder einer verbundenen Kamera importiert werden. Für zusätzliche Informationen siehe unterstützte Datei-Formate.

    🔗Import ab Verzeichnis

    Du kannst entweder ein einzelnes Bild oder einen Ordner mit Bildern importieren durch Anwahl der zugehörigen Schaltfläche. Das/die importierte(n) Bild(er) werden automatisch einer Filmrolle zugeordnet, die den gleichen Namen hat wie das Verzeichnis, von dem aus sie importiert werden.

    Klicke auf “Bild…” oder “Verzeichnis…” um einen Dateiwähl-Dialog zu öffnen. Navigiere durch das Dateisystem und wähle die Datei(en) zu importieren.

    🔗Import-Optionen

    Klicke auf das Label “Parameter” oder den Expander daneben, um die folgenden zusätzlichen Optionen, betreffend den Dateiimport zu öffnen.

    Alle Import-Optionen eines Importes werden gesichert als Modul-Voreinstellungen. Drücke den Modul-Reset-Knopf, um alle Einstellungen auf Standard zurückzusetzen.

    Verzeichnisse rekursiv importieren (nur für Verzeichnis-Import)
    Setze für diese Option das Häkchen, um Bilder im angewählten Verzeichnis und allen Unterverzeichnissen zu importieren. Die Option wird nicht empfohlen, um eine große Anzahl Bilder auf einmal zu importieren. Für den Importprozess muss darktable Miniaturbilder aller Bilder erzeugen, kann aber letztlich nur die Allerletzten im Cache behalten. Deshalb ist es besser, Bilder in kleineren Mengen zu importieren, um die Restriktionen der Leistung zu vermeiden, die dieser Prozess verursacht.
    JPEG-Dateien ignorieren
    Setze für diese Option das Häkchen, fallsJPEG Bilder im gleichen Ordner sind, die du nicht importieren willst. Diese Option wird normalerweise gebraucht, wenn die Kamera RAW+JPEG sichert und du nur an den RAW arbeiten, und die JPEG Dateien nicht bearbeiten willst.
    Ignoriere Exif-Bewertungen
    Setze für diese Option das Häkchen, um irgendwelche, in den Exifdaten der Bilder gespeicherten Bewertungen zu ignorieren und gebrauche den einstellbaren Vorgabewert (unten).
    Erstbewertung beim Importieren (darktable-Voreinstellungen)
    Erstbewertung mit Sternen (von 0 bis 5) für alle Bilder beim Import (Standard 1)
    Metadaten beim Import hinzufügen
    Setze hier das Häkchen, um automatisch beim Import Metadaten Felder und oder Tags zu setzen (siehe unten).
    Metadaten
    Wenn das Häkchen bei “Metadaten beim Import hinzufügen” gesetzt ist, wird eine Liste der sichtbaren Metadaten Felder gezeigt (für Details siehe Metadaten-Editor). Alle Inhalte der ausgefüllten Felder werden den importierten Bildern automatisch zugefügt. Du kannst auch Voreinstellungen wählen, die im Metadaten Editor gesichert sind.

    Doppelklick auf eine Eingabe um das zugehörige Feld um das Feld zurückzusetzen. Doppelklick auf das Feld “Metadaten Voreinstellungen” um alle Felder zurückzusetzen.

    Wenn darktable-Voreinstellungen>Speichern>für jedes Bild Begleitdatei schreiben nicht gesetzt ist, wird eine zweite Kolonne aufgezeigt “von XMP” präsentiert, sodass du den Import von nicht erwünschten Metadaten von xmp-Dateien verhindern kannst.

    Stichwörter
    Falls du zusätzliche Tags beim Import beifügen willst, kannst du diese hier in einer kommaseparierten Liste eintragen. So wie bei den Metadaten kannst du auch welche von den Voreinstellungen im Modul Verschlagwortung wählen.

    Wichtig: Beim Import von einem Verzeichnis wird darktable keine Duplikate deiner Bilder in ein separates Verzeichnis schreiben, sondern diese in situ bearbeiten. Ein Import wird lediglich Details dieser Bilder in die Datenbank von darktable schreiben (und wo nötig eine XMP Seitenwagendatei eröffnen), sodass die Bilder gesichtet und bearbeitet werden können.

    Das bedeutet auch, dass, falls du Bilder nach dem Import vom Datenträger löschen solltest, darktable diese nicht mehr finden kann: Bilder zu importieren bedeutet nicht eine zusätzliche . Sicherung deiner Verzeichnisse. Außerdem wird darktable keine Änderungen in deinen Verzeichnissen nachvollziehen. Irgendwelche zusätzlichen neuen Bilder werden erst in darktable sichtbar, nachdem diese importiert sind.

    🔗Import von einer verbundenen Kamera

    Falls eine Kamera entdeckt wird, dann wird diese automatisch im Importverzeichnis angezeigt. Falls du dann mit der Maus über dem Kamerabild bist, zeigt ein Pop-up-Fenster die Informationen der Kamera (Modell, Version der Firmware usw.). Je nach den Möglichkeiten der Kamera werden folgende Optionen eingeblendet:

    Import ab der Kamera:
    Ein Import Dialog wird eingeblendet, mit allen Bildern in dieser Kamera, die in eine Filmrolle importiert werden können.

    Das Wurzelverzeichnis, in dem importierte Bilder gesichert werden und die Namensgebung von Unterverzeichnissen können bestimmt werden in darktable-Voreinstellungen>Importieren.

    Tethering
    Tethering wird gebraucht um deine Kamera in darktable zu integrieren. Wenn du so Bilder mit deiner Kamera aufnimmst, werden automatisch in darktable importiert, sodass du direkt die Resultate der Aufnahme beurteilen kannst. Du kannst auch Aufnahmekriterien setzen wie, Anzahl Bilder, Zeitdifferenz zwischen Aufnahmen, wie auch Kameraeinstellungen, Verschlusszeit, Blendenöffnung usw.

    Falls von deiner Kamera unterstützt, wird die Anwahl von Tethering Aufnahme die Ansicht tethering öffnen.
