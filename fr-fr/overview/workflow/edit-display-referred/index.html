                    développer une image : flux de travail relatif à l’affichage

                    Ceci est un mode traditionnel. Il est conservé pour fournir une rétrocompatibilité avec les développements effectués dans les anciennes versions de darktable et pour permettre aux utilisateurs de continuer avec leur ancienne méthode de travail sans les forcer à utiliser le nouveau flux de travail relatif à la scène.

                    Le flux de travail relatif à l’affichage met davantage l’accent sur l’exécution du traitement d’image dans la partie non-linéaire relatif à l’affichage du pipeline graphique. Par défaut il utilise le module courbe de base pour mapper les images de l’espace linéaire relatif à la scène vers l’espace relatif à l’affichage, bien que d’autres outils de mappage des tonalités (tel que le module courbe des tonalités) puissent aussi être utilisés. De nombreux modules sont déplacés plus tard dans le pipeline graphique (après ce mappage des tonalités) de telle sorte qu’ils travaillent avec des valeurs de pixel (relatives à l’affichage) gamma-encodées plutôt qu’avec des valeurs de pixel linéairement-encodées (relatives à la scène).

                    La plupart des étapes de base requises pour développer des images dans le cadre du flux de travail relatif à l’affichage sont assez similaires à celles requises pour le flux de travail relatif à la scène. Les principales différences résident dans le choix des modules et dans leur ordre d’apparition dans le pipeline graphique. Pour voir la différence dans l’ordre des modules entre flux de travail relatif à l’affichage et flux de travail relatif à la scène, veuillez vous référer à la section ordre des modules.

                    Remarque : Cette section décrit un certain nombre de modules originels qui ne sont plus recommandés pour une utilisation dans le flux de travail relatif à la scène. Il est recommandé aux nouveaux utilisateurs de se référer à la section développement d’une image : flux de travail relatif à la scène pour obtenir un guide sur la meilleure façon de traiter les images dans darktable.

                    balance des blancs
                    Le module balance des blancs travaille de la même façon que dans le flux de travail relatif à la scène, par défaut, il utilise les coefficients de balance des blancs fournis par le boîtier. Si cela ne donne pas de résultats acceptables, utilisez les préréglages du boîtier ou déterminer la balance des blancs à partir d’un point neutre de votre image. Le curseur de température peut être utilisé pour rendre l’image “plus chaude” ou “plus froide”. Un étalonnage des couleurs plus avancé est laissé à d’autres modules, comme indiqué plus loin.
                    correction exposition
                    Le module exposition travaille de la même façon que dans le mode relatif à la scène, mais la façon dont vous l’utilisez est un peu différente. Dans le mode relatif à l’affichage, vous devez vous assurer de ne pas trop affaiblir vos hautes lumières, et utiliser le module courbe de base pour ajuster les tons moyens si nécessaire.

                    Bien que vous puissiez utiliser le module exposition pour ajuster le niveau de noir pour fournir plus de contraste, vous devez faire très attention en faisant cela car vous pouvez vous retrouver avec des valeurs RVB négatives. Il est préférable d’augmenter le contraste en ajustant le pied de la courbe de base, cependant cela peut être un peu délicat et c’est l’une des raisons pour lesquelles le module filmique rvb a été introduit dans darktable.

                    réduction du bruit
                    Comme pour le flux de travail relatif à la scène, le meilleur point de départ pour la réduction du bruit est le module réduction du bruit (profil). De même, vous pouvez aussi choisir d’utiliser les modules réduction du bruit raw, flou de surface, réduction du bruit astro photo, ou le module égaliseur de contraste.
                    fixing spots
                    As with the scene-referred workflow, you can use the retouch and hot pixels modules to correct artifacts in your image.
                    corrections géométriques
                    Comme pour le flux de travail relatif à la scène vous pouvez utiliser les modules recadrer et pivoter, correction de perspective et correction des objectifs pour corriger les distorsions et recadrer votre image.
                    récupérer des détails
                    Les images raw contiennent souvent plus d’informations que vous ne pouvez en voir à première vue, en particulier dans les ombres. Le module ombres et hautes lumières aide à ramener ces détails dans des valeurs de tonalité visibles. De par la nature du capteur numérique, les détails structurels dans les hautes lumières totalement brûlées ne peuvent pas être récupérés. Cependant, vous pouvez corriger les dominantes de couleurs inopportunes dans ces zones avec le module reconstruire hautes lumières. Le module reconstruction des couleurs est capable de remplir les zones surexposées avec des couleurs appropriées basées sur leur environnement. Le module filmique rvb offre aussi une reconstruction des hautes lumières, mais veillez à désactiver tout d’abord le module courbe de base.
                    ajustement des tonalités
                    Presque tous les flux de travail sont susceptibles d’inclure l’ajustement de la plage des tonalités de l’image et darktable propose plusieurs modules pour aider à faire cela. Le plus basique est le module contraste luminosité saturation. Dans le module courbe des tonalités, les tonalités sont ajustées en construisant une courbe. Les modules niveaux et niveaux rvb proposent une interface concise, avec trois marqueurs dans un histogramme. Et bien sûr, rien ne vous empêche d’utiliser le module filmique rvb dans un flux de travail relatif à l’affichage si vous le souhaitez (après avoir désactivé le module courbe de base).
                    amélioration du contraste local
                    L’amélioration du contraste local peut accentuer les détails et la clarté de votre image. Utilisé avec soin, il peut donner à votre photographie le bon piqué. Plusieurs modules sont disponibles pour cette tâche. Le module contraste local est aisé à manipuler, avec juste quelques paramètres. Une technique beaucoup plus polyvalente, mais aussi plus complexe, est offerte par le module égaliseur de contraste. Jetez un œil à ses préréglages pour avoir une idée de son fonctionnement. Le module égaliseur de contraste est le “couteau suisse” de darktable pour de nombreux ajustements où la dimension spatiale joue un rôle. Notez que l’emplacement de ce module dans le pipeline graphique diffère significativement entre les flux de travail relatif à la scène et relatif à l’affichage.
                    ajustements des couleurs
                    darktable propose de nombreux modules pour ajuster les couleurs d’une image. Une technique très simple est implémentée dans le module correction des couleurs. Utilisez ce module pour donner à l’image une teinte globale ou pour ajuster la saturation globale des couleurs. Le module zones de couleurs offre un contrôle beaucoup plus fin pour ajuster la saturation, la luminosité et la teinte dans les zones définies par l’utilisateur. Le module courbe des tonalités – en plus de l’ajustement classique des tonalités – vous donne un contrôle précis des couleurs d’une image. Enfin, si vous avez l’intention de convertir une image en noir et blanc, un bon point de départ, avec une interface utilisateur simple d’utilisation et intuitive, vous est proposé par le module monochrome. Vous pouvez également envisager d’utiliser le module calibration des couleurs.
                    netteté
                    Si vous démarrez votre flux de travail à partir d’une image raw, vous devrez augmenter la netteté de la sortie finale. Le module renforcer la netteté peut le faire avec l’approche classique USM (unsharp mask), disponible dans la plupart des logiciels de traitement d’image. Un autre moyen très polyvalent d’améliorer les bords d’une image est offert par le module filtre passe-haut, en combinaison avec le riche ensemble d’opérateurs de fusion de darktable.
                    effets artistiques
                    darktable est livré avec un riche ensemble de modules d’effets artistiques. Par exemple, vous pouvez utiliser le module filigrane pour ajouter un filigrane à votre image. Le module grain simule le bruit typique des images analogiques classiques. Utilisez le module mappage des couleurs pour transférer l’aspect et le ressenti d’une image couleur sur une autre. Le module faible lumière vous permet de simuler la vision humaine pour rendre les images en faible luminosité plus proches de la réalité. Le module filtre dégradé ajoute un dégradé neutre ou coloré à votre image pour l’exposition et la correction des couleurs.
